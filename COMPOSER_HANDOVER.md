COMPOSER v0.2（単一HTML）の続き開発です。

【追加機能】
1. UNDO/REDO
   - Ctrl+Z / Ctrl+Shift+Z
   - キーフレームのadd/remove/updateをhistoryスタックに積む
   - 上限50ステップ

2. トランスフォームショートカット（AE準拠）
   - P: X/Y位置のみ表示/非表示トグル
   - S: スケールのみ表示/非表示トグル
   - A: アンカーポイントのみ表示/非表示トグル
   - U: キーフレームがあるプロパティのみ表示
   - ショートカットはインプットフォーカス時は無効

3. オーディオ
   - BGM/SEファイル読み込み（WAV/MP3/OGG）
   - Web Audio APIで再生、映像と同期
   - Mキーで現在フレームにマーカー追加
   - タイムラインにオーディオトラック行を追加
   - マーカーはタイムライン上に表示

4. サムネ縦幅をトラック高さに動的追従
   - .tl-frame-block の height をトラック高さに追従
   - トラック高さ = (timeline高さ - kf-row) / トラック数
   - 現状は固定32pxなので動的に計算するよう変更

【デザイントーン】
ANIMATORと統一のtDRスタイル（--acid:#36FF00、モノスペース、黒基調）

【既存仕様】
- ANIMATOR_v1 JSONインポート
- X/Y/Scale/AX/AY キーフレーム（線形補間）
- ビューポートドラッグでオートキー
- アンカーハンドル（×）と位置ハンドル（□）
- 4K PNG連番書き出し（JSZip）
- タイムラインサムネ・ルーラー・スクラブ
